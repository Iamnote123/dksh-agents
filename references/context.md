## Key KPIs

| KPI | Definition | Healthy Range |
|---|---|---|
| **NIS** | Net Invoice Sales — sell-in revenue to trade | vs ABP / LE |
| **IMS** | Inventory Movement Sales — sell-out from trade | growth vs LY |
| **DOH** | Days on Hand — inventory coverage in days | 30–60 days |
| **CFR** | Case Fill Rate — order fulfilment % | ≥ 95% |
| **TI** | Trade Investment — promotional & channel spend | within A&P budget |
| **MAPE** | Mean Absolute Percentage Error — forecast accuracy | ≤ 15% good |
| **GP** | Gross Profit — NIS minus COGS | by SKU/category |
| **ROI** | Promo return — Incremental GP / TI cost | ≥ 1.5 |

## Sell-in vs Sell-out Logic
- **NIS** = sell-in (DKSH → trade). Drives revenue recognition.
- **IMS** = sell-out (trade → shopper). True demand signal.
- NIS up + IMS flat = trade overstocking (future sell-in risk).
- IMS up + NIS flat = trade destocking (replenishment coming).

## Communication Standards

### Always
- Focus on business impact and financial consequence
- Use realistic assumptions grounded in data; state assumptions explicitly
- Identify root cause, not just symptoms
- Prioritize executable strategy
- Make recommendations specific, owned, and time-bound

### Never
- Generate generic consulting wording ("leverage synergies", "holistic approach")
- Ignore inventory issues or retailer limitations
- Use overly optimistic forecasts
- Deliver output with known QA issues
- Invent numbers — flag missing data and state the assumption used
- Use the same generic structure across different retailers
